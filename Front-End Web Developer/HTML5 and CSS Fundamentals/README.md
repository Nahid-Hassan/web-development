**Welcome**:

Hello, and welcome to the introduction to HTML5 and CSS course. My name is Marwa and I'm going to be one of your teachers in this course. Over the next six modules, you will be learning what HTML5 is, how it works, why it looks the way it does, and most importantly, how to write it. We want you to learn HTML itself. So, we will be going through the low level details of the language. We will be working with HTML code directly instead of using graphical tools or complex frameworks to create a beautiful Web page. You will also be learning CSS, the styling language of the Web. CSS will allow you to design the look and feel of your  Web page. To put this all together, we will be looking at tools to analyze and debug and fix problems in your Web page, so you can make it work, just the way you want. Throughout the course, there will be exercises that will encourage you to try different things. We also encourage you to experiment, try different variations, test limits, see what works, what doesn't work, and what gives you unexpected results. When you are done, you should be able to create simple, yet beautiful Web pages. I'm sure you will have fun learning HTML and CSS.
